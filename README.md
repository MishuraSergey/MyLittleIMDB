# LITTLE OMDB APP

Find useful information about movies and watch trailers

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Built with

- HTML
- CSS Flexbox
- Vue CLI (Vue 3, Webpack, Babel)
- SASS
- Third party API's: Youtube API, OMDB API
